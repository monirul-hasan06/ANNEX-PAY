About Annex Pay

Welcome to Annex Pay
Developed by Team Hash Include

ANNEX PAY is a banking system developed using the C programming language.
It is created by students of Bangladesh University of Business and Technology (BUBT).

This system allows users to:

Create bank accounts

Deposit and withdraw money

Transfer funds

Pay bills

Recharge mobile balance

Request loans and funds

An Admin Panel is also available where administrators can:

Manage user accounts

Approve loans

Apply profit

Perform other administrative operations
